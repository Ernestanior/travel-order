## 📝 表单字段说明

### 必填字段（带 * 标记）
1. **Customer Name** - 客户名称
2. **Tel / HP** - 电话号码

### 可选字段

#### 客户信息
- Booking Date（默认今天）
- Address - 地址
- Fax - 传真
- Discount - 折扣金额
- Staff - 负责员工

#### 航班信息
- Tour Code - 旅游代码
- Tour Description - 旅游描述

**Departure（去程）**:
- Date - 日期
- Time - 时间（如 0900/1525）
- Flight - 航班号
- Dest - 目的地

**Departure 2（转机）**: 同上

**Arrival（回程）**: 同上

**Arrival 2（转机）**: 同上

#### Items（费用项目）
每个 Item 包含：
- Item Name - 项目名称（如 Air Ticket, Airport Tax）
- Quantity - 数量
- Unit Price - 单价
- Total - 总价（自动计算）

**常见 Items**:
- Air Ticket
- Airport Tax
- Hotel
- Insurance
- Visa Fee
- Tour Guide Fee

#### Passengers（乘客）
- Passenger Name - 乘客姓名
- 支持格式：`MICHEAL TOO X 23 PAX`（表示23个同名乘客）

#### Special Instruction
- 特殊说明或备注

## ⚠️ 注意事项

1. **Booking Number 自动生成**
   - 用户无需手动输入
   - 后端自动从最后一个订单号+1

2. **日期格式**
   - 前端使用 HTML5 date input（YYYY-MM-DD）
   - 显示时可能需要转换为 DD/MM/YYYY

3. **时间格式**
   - 自由文本输入
   - 建议格式：`HHMM/HHMM`（如 `0900/1525`）

4. **金额计算**
   - Item Total = Quantity × Unit Price
   - Booking Total = Sum of all Items
   - 使用 JavaScript Number 精度，显示 2 位小数

5. **乘客格式**
   - 支持单个乘客名字
   - 支持批量格式：`NAME X N PAX`
   - 如：`MICHEAL TOO X 23 PAX` 表示 23 个同名乘客
