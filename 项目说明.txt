根据您提供的旧系统截图，我已经创建了一个完整的 Next.js + TypeScript 订单管理系统。

=====================================
主要功能模块
=====================================

1. Main Menu（主菜单）
   - 系统概览和统计数据
   - 快速进入各功能模块

2. Booking Order（预订订单）
   - 订单列表：显示 Booking#, Customer, Depart Date, Arrival Date, Total Cost, Paid, Outstanding, Status
   - 订单详情：客户信息、航班信息（支持4段航班）、乘客数据、收据记录、特殊说明

3. Exchange Order（换票订单）
   - 订单列表：显示 Exchange#, Agent, Dept Date, Arv Date, Total Cost, Paid, Outstanding, Status, Booking#
   - 订单详情：供应商信息、航班信息、乘客数据、换票项目明细、收据记录

4. Customer（客户管理）
   - 客户列表和搜索
   - 客户信息维护

5. Supplier（供应商管理）
   - 供应商（航空公司）列表
   - 供应商信息维护

6. Reports/Printouts（报表）
   - 13种不同的报表查询选项
   - 包括预订查询、换票查询、收据报表、利润报表等

=====================================
技术栈
=====================================

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Lucide React (图标)
- 响应式设计

=====================================
快速开始
=====================================

1. 安装依赖：
   npm install

2. 运行开发服务器：
   npm run dev
   
   然后打开浏览器访问：http://localhost:3000

3. 构建生产版本：
   npm run build

=====================================
部署到 Vercel（3分钟）
=====================================

方法一（最快）：
   npm install -g vercel
   vercel login
   vercel
   vercel --prod

方法二（通过 GitHub）：
   1. 推送代码到 GitHub
   2. 在 vercel.com 导入项目
   3. 点击 Deploy

详细步骤请查看：立即部署.md
