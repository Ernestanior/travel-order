## 📝 详细使用说明

### 编辑订单

1. **进入编辑模式**
   - 访问订单详情页
   - 点击右上角 "Edit" 按钮
   - 所有字段变为可编辑

2. **编辑基本信息**
   - Customer Name（必填）
   - Address
   - Tel/HP（必填）
   - Fax
   - Discount
   - Staff
   - Tour Code
   - Tour

3. **编辑航班信息**
   - Departure（去程）
   - Departure 2（转机，可选）
   - Arrival（回程）
   - Arrival 2（转机，可选）
   - 每个航段包含：日期、时间、航班号、目的地

4. **管理 Items**
   - 点击 "Add Item" 添加新项目
   - 填写：Item Name, Quantity, Unit Price
   - Total 自动计算
   - 点击垃圾桶图标删除 Item
   - 底部显示总金额

5. **管理 Passengers**
   - 点击 "Add" 添加新乘客
   - 输入乘客名字
   - 点击垃圾桶图标删除 Passenger

6. **编辑特殊说明**
   - 在 Special Instruction 文本框输入备注

7. **保存或取消**
   - 点击 "Save" 保存修改
   - 点击 "Cancel" 取消，恢复原数据

### 删除订单

1. **打开删除确认**
   - 在查看模式点击 "Delete" 按钮
   - 弹出确认对话框

2. **确认删除**
   - 阅读警告信息
   - 点击 "Delete" 确认删除
   - 或点击 "Cancel" 取消

3. **删除后**
   - 显示成功提示
   - 自动跳转到订单列表页

## ⚠️ 注意事项

### 当前限制

1. **Items 和 Passengers 编辑**
   - 只能在客户端临时修改
   - 保存时不会更新到数据库
   - 需要后续实现 Items 和 Passengers 的 API 更新

2. **Payments 不可编辑**
   - 编辑模式下隐藏 Payments
   - Payments 需要通过专门的 "Make Payment" 功能添加

3. **Financial Summary**
   - 编辑模式下隐藏
   - 金额计算基于当前 Items
   - 实际金额需要从数据库计算

### 建议优化

1. **Items 和 Passengers API**
   - 实现单独的增删改 API
   - 点击 Save 时同步更新

2. **实时保存**
   - 每个字段失焦时自动保存
   - 或添加"自动保存"提示

3. **更好的验证**
   - 字段级别的错误提示
   - 高亮显示错误字段

4. **撤销/重做**
   - 记录编辑历史
   - 支持撤销操作
