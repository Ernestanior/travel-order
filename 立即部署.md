# 🚀 立即部署到 Vercel

## 最快方法（3分钟）

### 步骤 1：安装 Vercel CLI

打开终端，运行：

```bash
npm install -g vercel
```

### 步骤 2：进入项目目录

```bash
cd /Users/ern/Desktop/code/airline-order
```

### 步骤 3：登录 Vercel

```bash
vercel login
```

会打开浏览器让你登录，可以用 GitHub、GitLab 或 Email 登录。

### 步骤 4：部署

```bash
vercel
```

第一次会问几个问题，**全部按回车使用默认值即可**：

```
? Set up and deploy "~/Desktop/code/airline-order"? [Y/n] Y
? Which scope do you want to deploy to? (你的账号)
? Link to existing project? [y/N] N
? What's your project's name? airline-order (直接回车)
? In which directory is your code located? ./ (直接回车)
? Want to modify these settings? [y/N] N (直接回车)
```

等待部署完成，会给你一个预览链接，例如：
```
https://airline-order-xxx.vercel.app
```

### 步骤 5：部署到生产环境

```bash
vercel --prod
```

完成！你会得到一个生产环境链接。

## 更稳定的方法（通过 GitHub，5分钟）

### 步骤 1：推送代码到 GitHub

```bash
cd /Users/ern/Desktop/code/airline-order

# 初始化 Git
git init

# 添加所有文件
git add .

# 提交
git commit -m "Initial commit"

# 在 GitHub 上创建新仓库
# 然后添加远程仓库（替换为你的仓库地址）
git remote add origin https://github.com/你的用户名/airline-order.git

# 推送
git push -u origin main
```

### 步骤 2：在 Vercel 上部署

1. 访问 https://vercel.com
2. 点击 "Sign Up" 或 "Login"（用 GitHub 账号登录）
3. 点击 "Add New" → "Project"
4. 从列表中选择你的 `airline-order` 仓库
5. 点击 "Import"
6. 保持默认设置，点击 "Deploy"
7. 等待 1-2 分钟，部署完成！

## 设置自定义域名（可选）

1. 在 Vercel 项目中，点击 "Settings" → "Domains"
2. 输入你的域名（例如：orders.yourcompany.com）
3. 按照提示在你的域名服务商添加 DNS 记录
4. 等待 DNS 生效（通常几分钟到几小时）

## 常见问题

**Q: 部署失败怎么办？**
A: 查看错误信息，通常是依赖安装问题。可以运行 `npm install` 和 `npm run build` 本地测试。

**Q: 免费吗？**
A: Vercel 免费计划足够个人和小型项目使用。每月 100GB 带宽。

**Q: 数据会保存吗？**
A: 当前版本使用模拟数据，刷新会丢失。需要连接数据库才能持久化。

**Q: 如何更新网站？**
A: 修改代码后，重新运行 `vercel --prod`，或推送到 GitHub（会自动部署）。
